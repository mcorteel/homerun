# Homerun

## Installation
1. Copy `config/config.sample.php` to `config/config.php`
2. Edit the variables inside `config/config.php` to match your own configuration
3. Apply all SQL diffs in `config/db_schema`
4. Run `bower install`
5. You should be all set
